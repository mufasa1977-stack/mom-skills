# Cross-Domain Transfer Canon

Locked 2026-05-23. The polymath engine — how the most creative thinkers borrow from unrelated fields.

## The Polymath Pattern

Aristotle (biology + ethics + politics + physics) → Da Vinci (painting + anatomy + engineering + flight) → Franklin (politics + science + journalism + diplomacy) → Edison (invention + business + lab) → Fuller (architecture + math + philosophy) → Jobs (design + tech + calligraphy) → Musk (physics + engineering + finance) → Munger (psychology + investing + history). **The method**: take a discipline's METHOD (not content), apply it to a domain that lacks it.

## The Science

- **Root-Bernstein** *Sparks of Genius* (1999) — 13 thinking tools all creatives use: Observing, Imaging, Abstracting, Recognizing Patterns, Forming Patterns, Analogizing, Body Thinking, Empathizing, Dimensional Thinking, Modeling, Playing, Transforming, Synthesizing.
- **Hofstadter** *Surfaces and Essences* (2013) — analogy is the engine of cognition. Surface vs deep features.
- **Gentner** structure mapping — map RELATIONS not OBJECTS. Solar system → atom works because both involve "orbiting." Check 6+/8 relations before transferring.
- **Fauconnier-Turner** *The Way We Think* (2002) — conceptual blending 4-space model. Two mental spaces blend, emergent structure in the blend.
- **Star-Griesemer** boundary objects — concepts that work across disciplines. Pentagon UFO files are boundary objects (science + military + politics + public).
- **Range vs Depth** — Epstein 2019 vs Ericsson. Kind environments reward specialization; wicked environments reward range. Resolution: BOTH (T-shape or comb-shape).

## The 10-Domain Forced-Lens Method (use when stuck)

1. State the problem in one sentence
2. Run it through 10 deliberately unrelated lenses (one per domain in the 11-stack above)
3. Write one sentence per lens
4. Pick the 2-3 most surprising candidates
5. Validate via structure-mapping (Gentner — does it map 6+/8 relations?)
6. Apply OR build a new canon skill

**Worked example**: "Witness testimony reaction shot feels stagey" → Film: too symmetric. Music: missing rest before the line. Theater: wrong distance — McCandless 45° rule. Magic: misdirection on the wrong word. Psychology: not Kahneman System 1 trigger word. **Resolution: shift cut on dialogue trigger word + asymmetric 45° framing.**
